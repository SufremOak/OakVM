OakVM Documentation

Welcome to the OakVM documentation. OakVM is a versatile virtual environment that supports multiple programming paradigms and languages.

Features

  • C++ API support
  • WebAssembly integration
  • Custom Oak Assembly language
  • Cross-platform compatibility

OakVM is an open-source project licensed under the MIT License.

WebAssembly Integration

Loading WASM Modules

// Example WebAssembly integration
OakVM vm;
vm.loadWasmModule("module.wasm");
vm.callWasmFunction("main");

Memory Interface

OakVM provides a linear memory model compatible with WebAssembly specifications.

Oak Assembly Language

Basic Instructions

PUSH 42     ; Push value onto stack
POP         ; Pop value from stack
ADD         ; Add two top values
STORE name  ; Store value in variable
LOAD name   ; Load variable onto stack

Control Flow

JUMP label  ; Unconditional jump
JMPIF label ; Conditional jump
CALL func   ; Call function
RET         ; Return from function
